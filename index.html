grid overlay javascript plugin

A raw JavaScript plugin that creates an overlay on your site for easy alignment of design elements to your grid during development.

Include the Code

Grab the file/code from Github and call the file at the end of your document:

<script src="javascripts/gridOverlay.js"></script>
</body>
</html>

Demo

Toggle the grid visibility with Option + G (Mac) or Alt + G (PC) Try it on now to see.

Options

The JS file includes variables that can be reset so you can specify page width, column width, gutter width, baseline height and whether or not the grid is visible on page load or has to be toggled on.
