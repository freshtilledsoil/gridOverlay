Grid Overlay jQuery Plugin

A jQuery plugin that creates an overlay on your site for easy alignment of design elements to your grid during development.

How to Use

  1. Grab the file/code from Github and call the file at the end of your document, after your jQuery call:
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js"></script>
<script>window.jQuery || document.write('<script src="javascripts/jquery.min.js"><\/script>')</script>
<script src="javascripts/gridOverlay.js"></script>
</body>
</html>
  2. Toggle the grid visibility with Ctrl + G (try it on this page to see a demo).
  3. Options: the JS file for the plugin uses variables to allow you to easily specify the page width, column width, gutter width, baseline height and whether or not the grid is visible on page load or has to be toggled on.
  4. Support: this is intended for development only, not for use in production sites, so support has only been tested in the most recent versions of all major browsers. If you want to test in older browsers and create support, feel free to submit a pull request.
